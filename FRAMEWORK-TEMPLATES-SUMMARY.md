## 🚀 How to Use Right Now

### Copy & Paste Method (Fastest!)

1. **Open:** `tcp-platform/public/templates/EXACT-SPARK-TEMPLATE.json`

2. **Copy all the text** (Ctrl+A, Ctrl+C)

3. **Go to Admin Panel:**
   - Navigate to your framework chapter
   - Click the framework step
   - Click "Add Page" or edit existing page
   - Click "Edit Content"

4. **Paste** (Ctrl+V)

5. **Save**

6. **Done!** You'll see the exact same design as your screenshot

## ✏️ To Create Different Framework

### Example: Create "VOICE" Framework

1. **Open:** `BLANK-FRAMEWORK-TEMPLATE.json`

2. **Change these fields:**
```json
"frameworkCode": "VOICE",
"frameworkTitle": "The VOICE Framework",
"frameworkLabel": "FRAMEWORK: VOICE",
"accentColor": "#3b82f6",
"backgroundColor": "#eff6ff",
```

3. **Update the letters:**
```json
"letters": [
  { "letter": "V", "meaning": "Validate Experience" },
  { "letter": "O", "meaning": "Open Channels" },
  { "letter": "I", "meaning": "Identify Solutions" },
  { "letter": "C", "meaning": "Create Plans" },
  { "letter": "E", "meaning": "Empower Growth" }
]
```

4. **Save & use in admin panel**
